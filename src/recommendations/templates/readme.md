# 

This folder contains reusable report templates for stakeholder communication.

## How to Use

1. Copy the template you need
2. Replace {{PLACEHOLDERS}} with actual data
3. Save to `reports/` folder with date

## Available Templates

| Template | Purpose | When to Use |
|----------|---------|-------------|
| `executive_summary.md` | 1-page overview 
| `stakeholder_report.md` | Detailed analysis 
| `action_plan.md` | Task tracking |
| `email_update.md` | Weekly team email |

## Placeholder Guide
- `{{DATE}}` - Current date
- `{{REVENUE}}` - Total revenue
- `{{PROFIT}}` - Total profit
- `{{INSIGHT}}` - Key finding
- `{{ACTION}}` - Recommended task
